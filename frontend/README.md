Miska Pho - Customer Frontend:

This is the primary customer-facing application for Miska Pho, a modern Vietnamese restaurant. 
It provides a browse the authentic menu, manage their shopping cart, and book tables online.

Live Website:

The customer application is currently deployed at:
https://miska-pho-frontend.vercel.app/

Tech Stack:

Frontend: React.js 
Styling: Tailwind CSS
Navigation: React Router DOM
Payment Integration: Stripe
State Management: React Context API 
Data Fetching: Axios
