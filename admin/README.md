Miska Pho - Admin Dashboard:

This is the administrative control panel for the Miska Pho restaurant management system.
This dashboard allows managers to handle the menu, categories, orders, and table bookings in real-time.

Live Deployment:

The admin panel is currently deployed and functional at:
https://miska-pho-admin.vercel.app/

Tech Stack:

Frontend: React.js
Styling: TailwindCSS 
Routing: React Router DOM v6
State Management: React Hooks 
Notifications: React Toastify
API Client: Axios
